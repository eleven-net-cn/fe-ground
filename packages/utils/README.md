## Documentation

http://fe-ground.eleven.net.cn/#/utils/

## Installation

```bash
# Installation
yarn add @e.fe/utils
# OR
npm i @e.fe/utils
```

## Usage

```ts
// Example
import { random } from '@e.fe/utils';
```

## 按需加载

@e.fe/utils 的 JS 代码默认支持基于 ES modules 的 tree shaking。
